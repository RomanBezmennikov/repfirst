В init организовать импорты данных так чтобы функции не перекрывали друг друга
(Модули разделите по смыслу, т.е. один модуль к примеру содержит функции, другой какие либо значения
на ваше усмотрение) объем файлов не важен, может быть по паре функций в каждом модуле и этого будет достаточно

Так же создайте файл который будет импортировать данный пакет в себя и используйте все обозначенные функции
внутри данного файла, чтобы запуская данный файл все импорты, и запуски функций производились корректно
